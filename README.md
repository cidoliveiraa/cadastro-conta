# cadastro-conta
Simple customer registration for credit.

Project made with Angular, .NET Core and MongoDB.

1. [ Requirements. ](#req)
2. [ Front-End. ](#api)

## Requirements
Node.js 6.x, SDK Docker, Docker-Compose

### Run Docker (raiz do projeto)
```
docker-compose up
```
